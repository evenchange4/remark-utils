# Remark utils

> Reusable utils to build markdown service power by [remarkjs](https://github.com/remarkjs/remark) and [elasticlunr](https://github.com/weixsong/elasticlunr.js).

## Installation

```bash
$ yarn add remark-utils
```

## Development

- node 11.1.0
- yarn 1.12.3

```bash
$ yarn install --pure-lockfile
$ npm start
```

## Test

```bash
$ yarn run format
$ yarn run eslint
$ yarn run flow
$ yarn run test:watch
$ yarn run build
```

## Publish

```bash
$ npm version patch
$ git push
$ npm run changelog
```
